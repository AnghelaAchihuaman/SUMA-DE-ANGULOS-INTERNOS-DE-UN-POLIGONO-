Idea Central

La suma de los ángulos internos de un polígono depende del número de lados que tiene. Esta suma se obtiene dividiendo el polígono en triángulos.

Clave Matemática

Para un polígono con N lados, la suma de los ángulos internos se calcula así:

S = ( n − 2 ) × 180°

N : número de lados del polígono.
S : suma total de los ángulos internos.

Tres Ángulos De La Vida

 TRIÁNGULO EQUILÁTERO  

                       S = ( N - 2) x 180°                               

S = ( 3 - 2) x 180°

S = 1 x 180°

El Refugio Rectangular

 CUADRADO 

                S= ( N - 2 ) x 180°   

                  S= ( 4 - 2 ) x 180°

                     S= 2 x 180°

                       S = 360°  ------ Rpta
